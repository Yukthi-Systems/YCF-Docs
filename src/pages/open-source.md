# Open Source

YCF handles other people's mail before it ever reaches an inbox — that's not something
we think should run as a black box. The plugin chain, the spoof/SPF/DKIM logic, the
exact SQL behind every policy decision — all of it is here to read, run, and change.

## License

YCF is licensed under the **GNU General Public License v3 (GPLv3)**. In short:

- You're free to use, modify, and self-host it, for personal or commercial use.
- If you distribute a modified version, it has to stay open under the same license —
  improvements flow back instead of getting locked away in someone's fork.

See the full license text in [`LICENSE`](https://github.com/Yukthi-Systems/Yukthi-Content-Filter-Docs/blob/main/LICENSE).

## Contributing

Issues and pull requests are welcome. See the [Contributing guide](/docs/contributing)
for the plugin-authoring workflow and code style notes. Start with an issue if you're
planning something non-trivial — especially anything touching plugin ordering or spam
scoring, where a wrong default can affect a lot of mail at once.

## Rewards for serious contributors

Significant contributions don't go unnoticed. If you land a meaningful feature, fix
something that's been genuinely painful, or keep up a sustained pattern of solid
contributions, reach out — we recognize and reward that kind of work.

:::note
The specifics of the rewards program (criteria, process, how to reach us) aren't
finalized yet — check back here, or ask in GitHub Discussions.
:::
